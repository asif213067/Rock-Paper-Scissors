# ✊✋✌️ Rock Paper Scissors

A responsive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Challenge the computer in a fun and interactive game with real-time results, score tracking, and a clean, user-friendly interface.

---

## 📌 Overview

This project is a browser-based implementation of the classic Rock Paper Scissors game. It demonstrates fundamental JavaScript concepts such as DOM manipulation, event handling, random number generation, conditional logic, and state management while delivering an engaging user experience.

---

## ✨ Features

* 🎮 Play against the computer
* 🎲 Random computer choice generation
* 🏆 Real-time win, lose, and draw detection
* 📊 Live score tracking
* 🔄 Play unlimited rounds
* 📱 Fully responsive design
* 🎨 Clean and modern user interface

## 🎮 How to Play

1. Choose **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects its move.
3. The winner is determined using the standard game rules:

   * ✊ Rock beats Scissors
   * ✋ Paper beats Rock
   * ✌️ Scissors beats Paper
4. The score updates automatically after each round.
5. Keep playing as many rounds as you like!
